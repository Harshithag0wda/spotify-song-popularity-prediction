# Spotify Song Popularity Prediction using Machine Learning

This project predicts the popularity of Spotify songs using machine learning techniques.

## Project Overview
The dataset contains various audio features of songs such as danceability, energy, loudness, tempo, and acousticness. These features are analyzed using Exploratory Data Analysis (EDA) and used to train a machine learning model to predict song popularity.

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. Model Training
5. Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset Features
- danceability
- energy
- loudness
- tempo
- valence
- acousticness
- popularity
